🌿 **核心理念**：不是“消灭自我”，而是**看清“自我”只是经验流中的一个念头**，从而减少对身份、信念的执着。

---

<h3 id="PDJK1">📅 **整体结构**</h3>
+ **第1–2天**：稳定注意力，建立“觉察者”基础  
+ **第3–4天**：弱化“我拥有经验”的自动绑定  
+ **第5–6天**：开放觉察，让经验自行显现  
+ **第7天**：整合体验，回归日常中的无我感

---

<h2 id="N78hI">🗓️ 每日练习指南</h2>
---

<h3 id="Gli8R">**Day 1：锚定呼吸，识别“观察者”**</h3>
+ **目标**：建立稳定的觉察锚点，初步区分“念头”与“觉察”。
+ **练习**（10分钟）：
    1. 静坐，闭眼，自然呼吸。
    2. 将注意力轻柔地放在鼻尖或腹部的呼吸感觉上。
    3. 当发现走神（念头、声音、情绪），**不评判**，只在心里轻声说：“哦，有念头”“有声音”，然后回到呼吸。
    4. 结束前问自己：“是谁在觉察呼吸？” **不回答，只感受问题本身**。
+ **关键提示**：不要试图“找到”观察者，只是**留意“觉察”本身的存在**。

---

<h3 id="B135z">**Day 2：身体扫描，消融“身体=我”**</h3>
+ **目标**：打破“我的身体”这一坚固认同。
+ **练习**（15分钟）：
    1. 平躺或坐稳，从脚趾开始，逐步将注意力移到身体各部位。
    2. 对每个部位，只观察感觉（冷/热/麻/无感），**不说“我的脚”**，只说“有冷感”“有压力感”。
    3. 当扫描到头部时，特别注意：**感觉只是感觉，没有“我在感觉”的附加层**。
    4. 结束时问：“身体在哪里结束？空间从哪里开始？” **保持开放，不求答案**。

---

<h3 id="eClpV">**Day 3：念头如云，不属“我”**</h3>
+ **目标**：将念头去个人化。
+ **练习**（12分钟）：
    1. 静坐，想象自己坐在湖边，天空中有云飘过。
    2. 每个念头升起时，**在心里标记为“思考”**，而不是“我在想XXX”。
    3. 试着把念头看作**天气现象**：不是“我的焦虑”，而是“有焦虑云飘过”。
    4. 结束时默念：“念头来，念头走，天空始终清澈。”

---

<h3 id="o0A6K">**Day 4：感官沉浸，无“我”之听/看**</h3>
+ **目标**：通过感官直接体验，绕过“自我解释”。
+ **练习**（15分钟，选一种）：
    - **听**：播放一段自然声音（雨声、鸟鸣），全神倾听，直到“听者”消失，只剩下声音本身。
    - **看**：凝视一支蜡烛或树叶，让视觉充满意识，**不命名、不评价**，只让“看”发生。
+ **关键**：当“我在听”的念头出现，轻轻放下，回到纯粹的感官输入。

---

<h3 id="dwDua">**Day 5：开放觉察，万物自显**</h3>
+ **目标**：从“聚焦”转向“开放”，让经验自行浮现。
+ **练习**（15分钟）：
    1. 静坐，不聚焦任何对象。
    2. 允许声音、身体感、情绪、念头自由进出意识，**如天空容纳万物**。
    3. 不抓取、不排斥，也不说“这是我的经验”。
    4. 如果“我”感强烈，问：“这个‘我’有形状吗？能被触摸吗？”
+ **提示**：这不是“放空”，而是**高度清醒的无中心觉察**。

---

<h3 id="RZcti">**Day 6：行走中的无我**</h3>
+ **目标**：将无我感带入动态生活。
+ **练习**（10–15分钟慢走）：
    1. 在安静处缓慢行走，每步3–5秒。
    2. 注意脚底接触地面的感觉、空气流动、光影变化。
    3. 当“我在走路”的念头出现，轻轻转向**纯粹的动作与感觉**。
    4. 想象：不是“我”在走，而是**宇宙在通过这具身体行走**。

---

<h3 id="lkAFB">**Day 7：整合——日常中的无我**</h3>
+ **目标**：将体验融入生活，不追求“特殊状态”。
+ **练习**（全天微练习）：
    - **喝水时**：只感受水的温度、流动，不说“我在喝水”。
    - **说话时**：留意语言自动流出，不预设“我要表达什么”。
    - **情绪升起时**：说“有悲伤”，而非“我悲伤”。
+ **晚间反思**（5分钟）：
    - 写下：“今天，哪些时刻‘我’消失了？经验本身是什么样子？”

---

<h2 id="rMwOz">🌟 温馨提示</h2>
+ **不要追求“无我”**：越想“达到无我”，“我”就越强。只需**如实觉察当下**。
+ **允许“失败”**：走神、自我感强烈都是练习的一部分。
+ **安全第一**：如有解离史、严重焦虑或精神疾病，请在专业人士指导下进行。
