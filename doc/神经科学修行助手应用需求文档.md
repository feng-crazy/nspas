# 神经科学AI修行助手需求文档

## 1. 系统概述

### 1.1 系统名称
神经科学AI修行助手

### 1.2 系统描述
基于神经科学原理的智能对话系统，可以分析用户思维过程、认知模式及行为倾向，提供当前神经机制解读、从神经科学解释用户的修行困惑，可以给出修行语句神经脑科学释义以及针对用户的需求个性化提供脑科学修行小工具。\n
## 2. 核心功能

### 2.1 用户思维、认知或行为的神经科学分析
- 分析用户的语言和思维过程，判断有哪些大脑区域或神经回路在运作，做实时神经机制解读
- 分析和判断用户的行为，触发大脑哪些机制与神经回路
- 从神经回路、脑科学角度解释用户目前面临的困境和思维反应，帮助用户认识自己的大脑

### 2.2 神经科学映射
- 将用户思维描述做脑科学机制、神经通路（如压力→HPA轴激活，决策困难→前扣带回功能减弱）的解释
- 将用户问的修行语录做脑科学与神经回路释义

### 2.3 个性化脑科学修行方案
- 根据用户需求，想要改变的神经回路，提供神经可塑性训练方法
- 提供正念、认知重构、神经可塑性训练等修行建议
- 根据用户的需求生成修行帮助小应用（HTML的动态页面，是个性化实时生成的HTML页面，用户确认之后，保存在用户的空间，用列表显示，点击可以重新加载出来），帮助用户认识和管理自己的思维和神经回路

### 2.4 历史对话记录查看
- 为每个用户提供历史对话记录查看功能
- 支持按时间顺序浏览过往对话内容
- 可查看神经科学分析结果和提供的修行建议
- 支持对话记录的搜索和筛选功能

### 2.5 Markdown格式支持
- 对话输出支持Markdown格式显示，包括标题、列表、链接、代码块等常用语法
- 用户输入的Markdown格式内容能够正确解析和渲染
- 修行方案和分析结果以格式化的Markdown形式展示，提升阅读体验
- 支持在历史对话记录中正确显示Markdown格式内容

## 3. 应用版本

### 3.1 Web页面
- 页面设计符合目标要求，营造平静氛围
- 支持对话输出和方案展示，兼容Markdown格式渲染
- 支持微信授权登录和账号密码登录
- 集成历史对话记录查看功能

## 4. 设计风格

### 4.1 配色方案
以深蓝和浅灰为主色调，辅以淡绿色点缀，营造宁静专业的大脑科学氛围

### 4.2 视觉细节
采用柔和的圆角设计，使用渐变阴影效果，图标采用线性风格，整体呈现科技感与亲和力的平衡

### 4.3 整体布局
首页上有三个入口，分别是神经科学分析、修行映射（神经科学映射）和修行小助手（个性化脑科学修行方案），采用卡片式布局，对话区域居中显示，功能模块分区明确，确保信息层次清晰易读，历史对话记录功能集成在用户个人空间中
