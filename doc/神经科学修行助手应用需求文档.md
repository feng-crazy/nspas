## 1. 核心目标
构建一个基于神经科学原理的AI对话系统，通过分析用户情感状态、认知模式及行为倾向，提供：

+ 实时神经机制解读（如杏仁核激活、前额叶调控等），从神经回路，脑科学角度解释用户目前面临的困境和思维反应。
+ 从神经科学，大脑回路以及区域系统连接角度理解修行和哲学相关语句
+ 提供权威神经科学知识库问答。
+ 个性化脑科学修行方案（正念/认知重构/神经可塑性训练）



## 2. 功能模块
### 2.1 智能对话引擎
+ **用户多轮情感分析**  
    - 实时识别文本中的情绪维度 
    - 结合上下文和对话历史追踪情绪演变轨迹
+ **神经科学映射**  
    - 将用户思维描述自动关联到神经通路（如压力→HPA轴激活，决策困难→前扣带回功能减弱）
    - 将修行语录做脑科学与神经回路释义
+ **动态知识调用**  
    - 基于LlamaIndex构建神经科学知识图谱（涵盖500+核心概念：默认模式网络、镜像神经元、BDNF等）神经科学最新论文等知识库。
    - 哲学与修行语录知识库
    - 调用联网搜索 MCP检索案例

### 3.1 后端服务（Go + Python混合架构）
![](https://cdn.nlark.com/yuque/__mermaid_v3/1ffe7af613d57e108b6269a0c3b5e256.svg)



+ **Go服务层**  
    - 负责高并发请求处理（Gin框架）  
    - 用户信息和会话管理
    - 路由转发给langchain微服务
+ **Python AI层**  
    - LangChain构建Agent：  
        * 对话问答agent
        * 修行方案计划agent
        * 使用国产大模型
        * 拥有长短期记忆功能
        * 能够调用联网搜索mcp
        * LangServe提供api服务
    - LlamaIndex实现：  
        * 神经科学文献向量化（PubMed/BioRxiv数据集）  
        * 多级检索策略（关键词+语义相似度+神经通路关联）
        * <font style="color:rgba(0, 0, 0, 0.9);">把 LlamaIndex 当“Retriever”挂到 LangChain 链</font>

### 3.2 关键组件
+ **智能问答组件** 
    - 调用公网的AI大模型，比如Qwen或者文心一言，结合知识库和对话历史相关信息对用户问题进行回答
    - Prompt模板：  
比如"你是一名神经科学与修行实践者，如何从神经脑科学角度，理解{用户输入}，下面信息可以提供一些语境信息，{知识库检索内容}，{搜索引擎提供的案例}，{用户对话历史}"
+ **修行方案计划生成器**  
    - 根据用户问题包含意图识别，给用户一些修行方案计划的建议
    - Prompt模板：  
比如"基于{用户情绪状态}和{神经机制}，生成包含{科学依据}的{天数}天训练计划"。
+ **问答与修行记录**
    - 记录用户的聊天历史， 修行打卡记录， 成长感悟记录。
