# 神经科学修行助手项目概要设计

## 1. 项目概述

### 1.1 项目名称
神经科学修行助手

### 1.2 项目目标
构建一个基于神经科学原理的AI对话系统，通过分析用户情感状态、认知模式及行为倾向，提供：
- 实时神经机制解读（如杏仁核激活、前额叶调控等）
- 从神经科学角度理解修行和哲学相关语句
- 提供权威神经科学知识库问答
- 个性化脑科学修行方案（正念/认知重构/神经可塑性训练）

### 1.3 核心价值
将日常思维反应给出对应神经脑科学的合理解释，让用户更了解自己和人性。通过「症状描述→脑科学解释→行为干预->重塑神经回路」闭环，区别于普通心理咨询AI。

## 2. 系统架构

### 2.1 整体架构
项目采用前后端分离的架构，包含微信小程序端、Web端和后端服务。

### 2.2 技术架构
- **前端**：微信小程序（原生组件）+ Web端（React/Vue）
- **后端**：Go + Python混合架构
  - Go服务层：负责高并发请求处理（Gin框架）、用户信息和会话管理
  - Python AI层：基于LangChain构建Agent、LlamaIndex实现知识检索
- **数据存储**：MongoDB存储用户记录

## 3. 功能模块

### 3.1 智能对话引擎
- 用户多轮情感分析
- 神经科学映射
- 动态知识调用

### 3.2 核心功能
- 思维症状-机制解释器
- 修行语录的神经科学解释器
- 交互式知识问答
- 修行方案生成器

### 3.3 修行记录功能
- 根据生成的修行方案计划智能生成表格
- 用户可以在上面填写完成情况以及感悟

## 4. 核心交付物
- 核心对话引擎+知识库MVP
- 双端对话UI+修行方案生成系统

## 5. 项目规划

### 5.1 阶段划分
- 第一阶段：核心对话引擎和知识库MVP开发
- 第二阶段：双端UI开发和基础功能实现
- 第三阶段：修行方案生成器和记录功能完善
- 第四阶段：测试优化和上线准备

### 5.2 预期成果
- 实现一个功能完整的神经科学修行助手应用
- 提供微信小程序和Web端两种使用方式
- 建立完善的神经科学知识库和修行方案库
