# 神经科学修行助手用户手册

## 1. 应用简介

神经科学修行助手是一个基于神经科学原理的AI对话系统，通过分析用户情感状态、认知模式及行为倾向，提供：
- 实时神经机制解读
- 从神经科学角度理解修行和哲学相关语句
- 权威神经科学知识库问答
- 个性化脑科学修行方案

## 2. 功能说明

### 2.1 智能对话

**功能描述**: 与AI助手进行对话，获得神经科学角度的解答和建议。

**使用方法**:
1. 在聊天界面输入您的问题或描述当前状态
2. 点击"发送"按钮或按Enter键
3. AI助手会从神经科学角度为您分析并给出建议

**示例问题**:
- "我最近总是感到焦虑，怎么办？"
- "为什么我'知而不行'？"
- "什么是神经可塑性？"

### 2.2 修行方案生成

**功能描述**: 根据您的当前状态生成个性化的神经科学修行方案。

**使用方法**:
1. 进入"修行计划"页面
2. 描述您当前的心理状态或遇到的挑战
3. 选择计划周期（7天、14天、21天或30天）
4. 点击"生成计划"
5. 系统会为您生成包含科学依据的训练计划

**计划内容**:
- 每日任务说明
- 神经科学依据
- 预期效果

### 2.3 修行记录

**功能描述**: 记录您的修行进展和感悟。

**使用方法**:
1. 选择已生成的修行方案
2. 每日完成修行任务后，记录完成情况
3. 写下当日的感悟和体会
4. 系统会保存您的记录，方便回顾和追踪

## 3. 使用技巧

### 3.1 如何获得更好的回答

- **详细描述**: 尽可能详细地描述您的情况，包括情绪、行为、想法等
- **提供上下文**: 在对话中提及之前讨论的内容，帮助AI更好地理解
- **追问**: 如果对回答有疑问，可以继续追问以获得更深入的解释

### 3.2 修行方案的使用

- **循序渐进**: 按照计划逐步完成，不要急于求成
- **记录感受**: 及时记录完成情况和感受，有助于追踪进展
- **灵活调整**: 如果某个任务不适合，可以根据实际情况调整

### 3.3 数据隐私

- 所有对话内容仅用于提供个性化服务
- 数据采用加密存储
- 您可以随时清除对话历史

## 4. 常见问题

### Q1: AI的回答不准确怎么办？

A: 您可以：
- 提供更详细的背景信息
- 重新表述您的问题
- 使用追问功能获得更深入的解释

### Q2: 如何清除对话历史？

A: 在聊天界面点击"清除历史"按钮即可。

### Q3: 修行方案可以修改吗？

A: 目前系统生成的方案是固定的，但您可以根据实际情况灵活执行任务。

### Q4: 数据会保存多久？

A: 您的对话历史和修行记录会一直保存，直到您主动删除。
